# The Annual Recycling Day

##
Se intento seguir el estilo de la imagen generada en una herramienta online para generar wireframes. 

## Header

Color de header verde para representar el reciclaje, junto con letras blancas para que haya un buen contraste. Para estilizar el header, se agrego una imagen png del logo de reciclaje.  

 
## Body
Se agrego una imagen de una persona pensando a lado del titulo "What can I recycle?" para mejor estilo.

Objetos que se pueden reciclar en boxes de diferentes colores, cada color representa lo siguiente
verde: vidrio
gris: orgánico
azul: papeles
rojo: metales
amarillo: plasiticos

Los containers se hicieron redondos y se les agrego un poco de shadow para que esteticamente se vean mejor. 

Se agregaron imagenes de fondo en cada container para que asi se pueda distinguir mejor las categorias.


## Footer
Color de footer verde para combinar con el header y letras negras para un contraste que no sea tan repetitivo. Icono de un camión para representar que pueden traer y llevar cosas.
